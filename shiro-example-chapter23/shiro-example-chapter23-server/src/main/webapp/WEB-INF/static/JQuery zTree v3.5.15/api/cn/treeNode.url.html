StringtreeNode.url

概述[ 依赖 jquery.ztree.core 核心 js ]

节点链接的目标 URL

1、编辑模式 (setting.edit.enable = true) 下此属性功能失效，如果必须使用类似功能，请利用 onClick 事件回调函数自行控制。

2、如果需要在 onClick 事件回调函数中进行跳转控制，那么请将 URL 地址保存在其他自定义的属性内，请勿使用 url

默认值：无

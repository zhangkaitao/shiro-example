Function(editable)zTreeObj.setEditable

概述[ 依赖 jquery.ztree.exedit 扩展 js ]

设置 zTree 进入 / 取消 编辑状态。

对于编辑状态的各种功能需要提前设置对应 setting 中的不同属性

请通过 zTree 对象执行此方法。

Function 参数说明

editableBoolean

true 表示进入 编辑状态

false 表示取消 编辑状态

返回值无

目前无任何返回值

function 举例

1. 设置 zTree 进入编辑状态

var treeObj = $.fn.zTree.getZTreeObj("tree");
treeObj.setEditable(true);
